- Uso: referência técnica do produto, não resposta teológica comum.
- Arquitetura: Next.js, páginas de seleção/personagem/chat/trilha, API de chat, base Markdown em knowledge.

---
# Lógica do chat

Cada pergunta deve passar por este filtro:

```
1. A pergunta pertence ao contexto desse personagem?
2. Existe base bíblica clara?
3. Qual texto bíblico sustenta a resposta?
4. Qual o significado teológico?
5. Qual aplicação prática é segura?
```

Se a pergunta fugir do contexto:

```
Essa pergunta pertence melhor ao contexto de [personagem].Posso responder conectando com o período atual ou te levar para esse mentor.
```
